# Friendly Architecture

## Frontend

The frontend will be developed using:

- HTML
- CSS
- Bootstrap
- JavaScript (minimal)

Responsibilities:

- Display pages
- Collect user input
- Show matches
- Show events

---

## Backend

The backend will be developed using:

- Python
- Flask

Responsibilities:

- Authentication
- User management
- Matching algorithm
- Database communication
- Validation

---

## Database

Database:

MySQL

Main tables:

- Users
- Interests
- User Interests
- Languages
- User Languages
- Events
- Event Participants

---

## Future Expansion

Possible future features:

- Messaging
- Notifications
- Mobile application
- AI recommendations
