# Friendly Project Plan

## Target User

The primary users are newcomers and international residents in Ireland who want to build local social connections through shared interests, languages and community activities.

## Core Problem

Newcomers may experience difficulty meeting people outside work, university or existing cultural communities.

## Core Solution

Friendly recommends relevant people based on shared interests, languages and city, while also helping users discover and join local community events.

## MVP Success Criteria

The MVP will be considered complete when a user can:

1. Register an account
2. Log in and log out
3. Create and edit a profile
4. Select interests and languages
5. Add a city
6. View recommended users
7. Understand why a user was recommended
8. Create a community event
9. Browse available events
10. Join or leave an event

## Technical Goals

- Build the backend using Python and Flask
- Store application data in MySQL
- Use relational database relationships
- Implement a weighted matching algorithm
- Validate user input
- Store passwords securely
- Use Git for version control
- Document the development process
