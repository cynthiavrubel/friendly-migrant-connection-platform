# Friendly Project Plan

## Product Vision

Friendly is a community integration platform that helps newcomers in Ireland build meaningful local friendships through shared languages, interests, location and welcoming community meetups.

## Target Users

The primary users are newcomers and international residents in Ireland who want to create a new social circle.

The platform is especially relevant to people who:

* Have recently moved to Ireland
* Do not yet have an established social group
* Prefer to meet people who speak the same language
* Feel more comfortable with people from a familiar cultural background
* Want to meet people through shared interests
* Want to attend events where participants are open to forming new friendships

## Core Problem

Newcomers may experience loneliness and difficulty meeting people outside work, university or existing cultural communities.

Many events and social groups are attended by people who already know each other. This can make joining alone uncomfortable for someone who has recently arrived.

## Core Solution

Friendly will:

* Recommend people based on shared interests, languages, city and relocation stage
* Help users understand why another person was recommended
* Support culturally and linguistically relevant connections
* Provide community meetups designed for people who want to build friendships
* Allow users to create, browse, join and leave meetups

## MVP Success Criteria

The MVP will be considered complete when a user can:

1. Register an account
2. Log in and log out
3. Create and edit a profile
4. Add a city
5. Select languages
6. Select interests
7. Select a relocation stage
8. View recommended users
9. Understand the reasons behind a recommendation
10. Create a community meetup
11. Browse available meetups
12. Join or leave a meetup

## Technical Goals

* Build the backend using Python and Flask
* Store application data in MySQL
* Design relational database relationships
* Implement a transparent weighted matching algorithm
* Validate user input
* Store passwords securely
* Use Git for version control
* Document the development process clearly

## MVP Boundaries

The MVP will not include:

* Real-time messaging
* Native mobile applications
* AI recommendations
* Payments
* Live location tracking
* Identity document verification
* Video calls
* Advanced moderation tools

## Expected Completion

Target completion: 1 September 2026

Final deadline: 5 September 2026
