# Friendly Design System

## Design Philosophy

Friendly should feel:

- Modern
- Minimalist
- Welcoming
- Calm
- Trustworthy
- Clean
- Accessible

The interface should help users feel comfortable and confident from the moment they arrive.

The design should never feel cluttered, childish or overwhelming.
